### 1. eclipse常用快捷键

1）代码块移动

上下移动（先选中代码块）

```
alt+上/下的方向键
```

左右移动（先选中代码块）

```
tab--向右移动
shift+tab--向左移动
```

2）查找类

```
shift+ctrl+r
```

注：在弹出的对话框中输入待查找到类。

![](/images/tools/eclipse/findresource.png)

3）返回上一次光标所在位置处

```
alt + 左右方向键
```

注：实现方法间的来回跳转，可以跨类文件跳转。
