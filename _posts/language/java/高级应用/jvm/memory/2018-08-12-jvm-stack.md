栈内存用来存储局部变量和方法调用。

1）栈帧

栈帧由三部分组成：局部变量区、操作数栈、帧数据区。

局部变量区和操作数栈的大小要视对应的方法而定，按字长计算。当调用一个方法时，从类型信息中得到此方法局部变量区和操作数栈大小，并据此分配栈内存，然后压入Java栈。
