Windows下，我们通常使用的vs,codeblcoks等IDE（集成开发环境）进行编译，调试，执行等操作。

集成开发环境（IDE，Integrated Development Environment ）是用于提供程序开发环境的应用程序，一般包括代码编辑器、编译器、调试器和图形用户界面工具。集成了代码编写功能、分析功能、编译功能、调试功能等一体化的开发软件服务套。
