### 查找jdbc驱动类为例

```
public static void main(String[] args) {
    ServiceLoader<Driver> serviceLoader=ServiceLoader.load(Driver.class);
    Iterator<Driver> drivers=serviceLoader.iterator();
    while(drivers.hasNext()){
        System.out.println(drivers.next());
    }
}
```
