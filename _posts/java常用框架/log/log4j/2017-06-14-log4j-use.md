### 代码中输出debug信息

以打印执行sql为例，输出sql语句

```
Logger log=Logger.getLogger(MybatisSqlPlugin.class);//成员变量

//方法中输出debug信息
public void test(){
    ...
    if(log.isDebugEnabled()){
        long end = System.currentTimeMillis();//执行完成后的结束时间
        long time = (end - start);
        String sql = getSql(configuration, boundSql, sqlId, time);
        log.debug(sql);
    }    
}

```
